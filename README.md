# 光环境策略生成工具

这是一个简单实用的室内光环境策略生成工具，输入空间类型和装修风格，即可快速获取专业的灯光设计建议。

## 功能特点
- 支持多种常见空间类型（客厅/卧室等）
- 适配主流装修风格（现代简约/新中式/北欧等）
- 输出结构化的灯光设计建议，包含主光源、辅助光源、重点照明和氛围营造

## 使用方法
1.  安装依赖：`pip install -r requirements.txt`
2.  运行程序：`python light.py`
3.  按照提示输入空间类型和装修风格，即可获取建议

## 示例
输入：
- 空间类型：客厅
- 装修风格：现代简约

输出：
1.  主光源：4000K中性光，亮度均匀
2.  辅助光源：吊顶灯带+防眩筒灯
3.  重点照明：电视墙、沙发背景墙
4.  氛围：干净、明亮、舒适放松
